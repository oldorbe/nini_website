# Legacy Decap CMS admin

This folder is the **Decap CMS** (formerly Netlify CMS) admin. It can be used with GitHub Pages + Cloudflare Worker OAuth; see [doc/deploy-decap-netlify.md](../doc/deploy-decap-netlify.md).

**Recommended:** Use the **Tina CMS** admin instead, deployed on Vercel: see [admin-tina/README.md](../admin-tina/README.md). The site frontend (GitHub Pages) and content (`content/*.json`) stay the same; only the editor app (Tina on Vercel) changes. Do not use both Decap and Tina to edit the same files.
