想做一个personal website/ 博客, 且需要便于用户(新手,无编程经验的人员)方便的发布管理内容 (也可以借助某种外部工具管理或者编辑内容,如果有的画)


内容分为三类型: 照片, 影片, 文字

# 1. 照片内容
* 参考https://www.myrelchernick.com/installations/ 或者 https://www.myrelchernick.com/images/
1. 我可能有一个或者2个tab, 每个tab都是一种类型的照片, 比如tab a是installation category(装置艺术), tab b是 images category (摄影)
2. 每个tab category里又有很多projects, 所以你会看见https://www.myrelchernick.com/installations/下有很多缩略图, 每个缩略图代表一个project
3. 你可以通过缩略图/或tab的下拉菜单子tab点入该项目页面(e.g. https://www.myrelchernick.com/work/installations/punchdrunk/). 进入后其实一个project是一个相册. 你可以通过左右箭头浏览上下一张图片, 也可以通过previous next button navigate到相邻project. 每个项目共享一个description text (比如'2005'), 绘制在图片下方
4. project第一个图片为缩略图

# 2. 影片内容
* 参考https://www.myrelchernick.com/videotapes/
1. 也是一个tab, 里面会有多个project, 点击缩略图或者下拉菜单可以进入具体project page. (e.g. https://www.myrelchernick.com/work/videotapes/esprit-descalier/)
2. video project page就是一个内嵌视频和一段description text
3. 缩略图要用户上传提供

# 3. 文字内容
* 参看 https://www.myrelchernick.com/texts/
1. tab page下也有缩略图 (default 为一个占位空白图片). 一样是点击缩略图或者下拉菜单可以进入具体project page
2. text project page 有图片 (optional), 文字. 文字里可以有附件下载链接内嵌(通常是pdf文件) ,也可以只是纯文字
3. 如果project page有图片,则第一张图片是缩略图. 如果没有, fallback到占位空白图片
